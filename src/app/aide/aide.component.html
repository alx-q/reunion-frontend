Bienvenue sur notre page de création de réunion! Voici quelques étapes pour vous guider:

  2. Entrez une brève description de l'évènement
  3. Ajouter un identifiant (8 lettres ou chiffres) pour ensuite permettre à vos collègues de fréquenter l'évènement
  4. Une fois le formulaire rempli, appuyez sur le bouton "Continuer" pour l'envoyer.
