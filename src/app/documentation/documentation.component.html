Le ngFor Dans Angular




La directive ngFor est une fonctionnalité puissante d'Angular qui permet de répéter un élément HTML ou un composant pour chaque élément d'un tableau ou d'une collection. Elle est utilisée pour afficher des données dynamiques de manière répétée.

Son utilisation est similaire à la boucle For.

NgFor Simple


Pour utiliser ngFor, il faut d'abord importer ou utiliser un tableau existant depuis le component.ts.

Exemple:
                            let items = [
                                "Item1"
                                "Item2"
                                "Item3"
                            ]

Ensuite, on peut l'utiliser dans le template d'un composant en utilisant la syntaxe suivante :


Dans cet exemple, 'element' représente l'élément HTML ou le composant que l'on souhaite répéter. 'items' est le tableau ou la collection contenant les données à afficher. 'item' est une variable locale qui représente chaque élément du tableau lors de l'itération. 'i' est une variable locale qui représente l'index de chaque élément.


NgFor d'un objet


On peut également utiliser ngFor avec des objets. 

                            let items = [
                                {{'{'}}
                                    key: "Alex",
                                    value: "hr"
                                {{'}'}},
                                {{'{'}}
                                    key: "John",
                                    value: "management",
                                {{'}'}}
                            ]

Dans cet exemple, 'items' est un objet et la pipe 'keyvalue' permet d'itérer sur les paires clé-valeur de l'objet.

Il est également possible d'ajouter des index, des compteurs et des conditions à l'aide de ngFor.

NgFor avec propriétés spéciales

Exemple

Vous pouvez aussi utiliser des propriétés spéciales de ngFor pour obtenir des informations supplémentaires sur l'itération en cours. Par exemple, vous pouvez utiliser "first" pour vérifier si c'est le premier élément, "last" pour vérifier si c'est le dernier élément, "even" pour vérifier si l'index est pair, et "odd" pour vérifier si l'index est impair.

Conclusion

En conclusion, la directive ngFor est un outil essentiel pour afficher des données répétitives dans Angular. Elle offre une grande flexibilité et permet de manipuler les données de manière dynamique.

Les Cartes Dans Bootstrap




C'est quoi une carte?


Une carte est un conteneur qui permet d'accueillir une multitude d'éléments, mais est composé principalement d'une image, de texte d'information brêves et un bouton d'action.
Elle est utilisée souvent pour afficher un bref résumé des articles d'un blogue ou d'une page de documentation comme celle-ci.



Implémentation

                        <div class="card" style="width: 18rem;">
                            <img src="..." class="card-img-top" alt="...">
                            <div class="card-body">
                                <h5 class="card-title">Lorem ipsum</h5>
                                <p class="card-text">Lorem ipsum dolor sit amet consectetur adipisicing elit. Eius similique cum aliquam.</p>
                                <a href="#" class="btn btn-primary">Go somewhere</a>
                            </div>
                        </div>

Exemples


Les cartes peuvent être implémentées de différentes façons, avec une image ou non, avec plus ou moins de texte ou avec ou sans bouton. Ils sont très flexibles, et parfaits pour les faire sur mesure.
